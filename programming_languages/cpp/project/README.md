## Conan Profile Issues

Be careful with the Conan profile because it has to be aligned with the
compiler and even with the libraries. Latest profile that works is:

```
[settings]
os=Linux
os_build=Linux
arch=x86_64
arch_build=x86_64
compiler=gcc
compiler.version=9
compiler.libcxx=libstdc++11
build_type=Release
[options]
[build_requires]
[env]
```
